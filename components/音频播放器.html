音频播放器 ​

  • 【默认】常规模式，可播放、暂停、调节播放进度、音量
  • 【可选】迷你模式，可播放、暂停、恢复、关闭、拖拽
  • 【设置】支持配速，默认[1, 1.5, 2]

Demo ​

vue
<template>
  <player miniable :list="list" :rates="[1, 2, 3]" />
</template>
<script setup lang="ts">
import player from './player/audio-player.vue'

  {
    name: '本地mp3',

</script>
