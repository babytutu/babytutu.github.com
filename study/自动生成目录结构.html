自动生成目录结构 ​

使用tree命令可以自动生成目录结构

bash
tree
1

安装 ​

Windows可以直接使用，cmd到目录执行即可

Mac需要安装tree命令

bash
brew install tree
1

常用参数 ​

排除目录 ​

排除依赖包目录

bash
tree -I node_modules
1

限制目录层级 ​

只生成2级目录结构

bash
tree -L 2
1

只显示目录 ​

bash
tree -d
1

防止中文乱码 ​

增加-N，支持中文命名的目录和文件

bash
tree -N
1

生成文件 ​

把tree的结果写入到一个md文件

bash
tree -N -I node_modules -d > list.md
1
